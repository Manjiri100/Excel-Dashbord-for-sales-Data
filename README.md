📊 Excel Dashboard for Sales Data

Project Overview:
This project demonstrates the creation of an interactive Excel dashboard for analyzing sales data. It provides clear insights into sales performance, trends, and key metrics, helping businesses make data-driven decisions quickly.

Key Features:

Interactive charts and graphs to visualize sales performance.

Pivot tables for summarizing data by region, product, and time period.

Conditional formatting to highlight top-performing products or regions.

Easy-to-read layout for quick insights and decision-making.
